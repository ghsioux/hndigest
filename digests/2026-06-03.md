## Top Stories

### [Gmail thinks I'm stupid, so I left](https://news.ycombinator.com/item?id=48375016)
A user abandoned Gmail due to the aggressive integration of AI features like "Tab to improve" and automated summaries that cannot be fully disabled. The HN discussion highlights that these features are likely driven by internal KPIs to boost LLM usage metrics rather than user demand, with many noting that the AI summaries frequently hallucinate critical details like delivery dates.

### [1-Click GitHub Token Stealing via a VSCode Bug](https://news.ycombinator.com/item?id=48371562)
A security researcher disclosed a vulnerability in VSCode that allows attackers to steal GitHub tokens via a malicious workspace extension. The researcher published this publicly out of frustration with Microsoft's Security Response Center (MSRC) silently patching bugs without credit. Commenters pointed out the inherent danger of browser-based IDEs holding global GitHub credentials instead of scoped, per-repo tokens.

### [CT scans of BYD car parts](https://news.ycombinator.com/item?id=48375824)
Lumafield published CT scans of BYD car components, revealing an extreme level of vertical integration not seen since early 20th-century Ford. BYD controls everything from the lithium mine to final assembly. While this drastically cuts costs, engineers in the comments note that highly integrated parts (like the E-axle) make aftermarket repairs nearly impossible, forcing expensive full-unit replacements.
