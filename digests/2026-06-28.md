### [Anonymous GitHub account mass-dropping undisclosed 0-days](https://news.ycombinator.com/item?id=48698617)
An anonymous GitHub account with no prior history began posting a stream of unreported exploit proof-of-concepts covering popular open-source tools, each claimed unfiled at time of release. The thread divided engineers over whether LLMs have fundamentally shifted the open-source security equation: some argued public code is now trivially bulk-scanned by anyone with a capable model, while others invoked Kerckhoffs and dismissed obscurity as a strategy that buys time at best.

### [DSpark: Speculative decoding accelerates LLM inference [pdf]](https://news.ycombinator.com/item?id=48696585)
DeepSeek released the DSpark paper detailing the speculative decoding system behind its 75 percent inference price cuts, deploying co-located lightweight draft models alongside production V4-Flash and V4-Pro. Readers tied the timing directly to the pricing drop and then settled into a familiar argument: U.S. frontier labs keep comparable optimizations proprietary, while export restrictions appear to have forced DeepSeek to squeeze every gain from whatever hardware it can actually buy.

### [IP Crawl: Living atlas of open webcams discovered on the public internet](https://news.ycombinator.com/item?id=48700834)
IP Crawl continuously scans public IP space for open webcam streams and publishes a browsable map of the results, from SCADA terminals and building lobbies to bedroom cameras and a deer feeder whose sign reads "Baiting deer is illegal!" The site drew a sharp privacy debate: some argued a publicly-accessible port is by definition not private, others called the site invasive regardless of technical legality, and the thread found no clean consensus.
