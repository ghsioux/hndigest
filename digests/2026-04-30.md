## Top Stories

### [HERMES.md in commit messages causes requests to route to extra usage billing](https://news.ycombinator.com/item?id=47952722)
Anthropic's Claude Code had a bizarre bug where the string "HERMES.md" in a commit message triggered an anti-abuse system, overcharging users. The situation escalated when an AI-generated support response refused to issue refunds for the technical error. Anthropic engineers eventually stepped in to apologize and issue credits, but the incident sparked intense discussion about the dangers of using AI for both billing logic and customer support.

### [Where the goblins came from](https://news.ycombinator.com/item?id=47957688)
OpenAI published a postmortem explaining why their latest models became obsessed with "goblins" and "gremlins." The culprit was an RLHF reward for "nerdy" metaphors in a specific persona, which unexpectedly bled into the model's general behavior. The community found it both amusing and alarming that the state-of-the-art fix for this alignment "whack-a-mole" was simply adding a directive to never talk about goblins to the system prompt.

### [We need a federation of forges](https://news.ycombinator.com/item?id=47948603)
Amidst growing frustration with GitHub's recent reliability issues, the creators of Tangled are proposing a federated alternative built on ATproto (the protocol behind Bluesky). The HN discussion debated whether complex federation is necessary when Git is already decentralized, with many pointing out that the real lock-in lies in the surrounding ecosystem of issues, pull requests, and CI/CD rather than the code itself.
