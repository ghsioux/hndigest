## Top Stories

### [Googlebook](https://news.ycombinator.com/item?id=48111545)
Google announced "Googlebook", a new line of laptops running an Android/ChromeOS hybrid built around Gemini. The announcement features an "AI mouse pointer" and deep Android integration. HN is highly skeptical, mocking the clumsy branding and predicting the hardware will be abandoned in a few years given Google's track record.

### [Bambu Lab is abusing the open source social contract](https://news.ycombinator.com/item?id=48109224)
Bambu Lab is blocking third-party slicers (like OrcaSlicer) from using their cloud API by checking the User-Agent, breaking remote printing for users not using the official client. The community is split: many are outraged by the walled-garden approach to hardware they own, while others argue Bambu shouldn't have to subsidize third-party cloud usage.

### [CERT is releasing six CVEs for serious security vulnerabilities in dnsmasq](https://news.ycombinator.com/item?id=48112042)
Six new CVEs were disclosed for dnsmasq, including heap overflows and infinite loops triggered by malicious DNS or DHCP packets. The vulnerabilities were discovered using AI-assisted fuzzing, prompting discussion about how AI tools are systematically breaking legacy C-based networking infrastructure.
